## 📌 حول المشروع

هذا المشروع يضم:

- ✅ إدارة بيانات الطلاب (CRUD operations)
- ✅ طبقات أمان متعددة (فقط الطبقات الاساسية)
- ✅ بناء API باستخدام C# و ASP.NET Core
- ✅ كود واضح وسهل الفهم لتعلم مفاهيم الأمان في الـ APIs
